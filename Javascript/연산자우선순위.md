## 연산자 우선 순위

| 우선순위 | 연산자                                                                          |
| -------- | ------------------------------------------------------------------------------- |
| 1        | ()                                                                              |
| 2        | new(매개변수 존재), .,[] (프로퍼티접근), ()(함수호출), ?.(옵셔널 체이닝 연산자) |
| 3        | new(매개변수 미존재)                                                            |
| 4        | x++, x--                                                                        |
| 5        | !x, +x, -x, ++x, --X, typeof, delete                                            |
| 6        | \*\* (이항 연산자 중에서 우선순위가 가장 높다)                                  |
| 7        | \*, /, %                                                                        |
| 8        | +, -                                                                            |
| 9        | <, <=, >, >=, in, instanceof                                                    |
| 10       | ==, !=, ===, !==                                                                |
| 11       | ??(null 병합 연산자)                                                            |
| 12       | &&                                                                              |
| 13       | \\\                                                                             |
| 14       | 삼항 연산자                                                                     |
| 15       | 할당 연산자 (=,+=,-=,...)                                                       |
| 16       | ,                                                                               |
